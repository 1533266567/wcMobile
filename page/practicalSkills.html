  • 波浪的形成

    波浪理論是嘗試以大自然的定律來融入金融市場中，本質上是希望捉摸宏觀的大自然韻律，應用在變化不定的微觀走勢中。由於市場的升降變化有如大海中的波浪，所以稱之為波浪理論。基本理論是以五個上升浪稱之為推動浪，而三個下跌浪則稱為調整浪。此時市況稱之為走了一個循環。

    但深入研究，每一組大浪尚可分為很多組小浪，而小浪亦可分為迷你浪……而難度較高的，在一組完整浪中隨時可額外殺出一個失敗浪。要辨別波浪的基本型態，就是第二浪的調整不會低於第一浪的起點，第三浪不會是主浪中最短的一浪，第四浪的浪底不會低於第一浪之浪頂。

    由於市場形態起伏不一，要使用波浪理論之時，首要是找出第三浪，其後則自然會分出其它四個浪來。要找出第三浪，先要認知第三浪為最具爆炸性之一浪。亦即是上升或下跌速度最快及最勁之一環節。最具爆炸性並不一定是最長，雖然後多時候的確是最長的一浪。但肯定不會是最短的一浪。而由於第三浪通常較長，所以亦會變化多端，經常會明顯地分成子浪及微浪，而其上升或下跌動力通常開始時較勁，其後又會轉趨平淡。

    要辨別波浪的基本型態，就是第二浪的調整不會低於第一浪的起點，第三浪不會是主浪中最短的一浪，第四浪的浪底不會低於第一浪之浪頂。

    以上顯示的是波浪的基本形態，五個上升浪可以作為一個大浪，而三個下跌浪亦可作為第二大浪，如此類推，向上可以進展至無限大的加大浪，向下亦可分割為無限小的子浪及微浪。而從日線圖的小浪中，可以成為小時圖的大浪，而小時圖亦可再如此類推分割下去

  • 延伸性

    一組完整的波浪，通常都具有延伸性，亦即是在三個主浪（一、三、五）的其中一個。亦即是此浪會再分為子浪及微浪，而長度則會明顯比其它兩個浪長。

  • 斜向三角形

    當波浪行走至第五浪之時，經常會出現一種斜向三角形，這是一種強弩之末的走勢；通常這種走勢又種之為消耗性的上升。是當大市經過一輪強力移動之後，在同一方向的走勢開始減慢。但由於曾大幅上升，定有基本之強勢因素支持，所以不易在短期內掉頭。大市仍在上升，但幅度轉為緩和，而波幅也收窄，此時便形成了一個斜向三角形，或稱為倒轉喇叭形。

  • 失敗浪

    雖然在理論上任何一點都有可能出現失敗浪，但通常在第五浪出現的機會較多。這是指第五浪的浪頂未能超越第三浪之浪頂。此時五個浪亦算是走完了。

  • 主浪定律

    1. 第二浪調整不會超越第一浪的起點。

    2. 第三浪不會是主浪中最短的一個浪，但亦不必是最長的。

    3. 在上升走勢，第四浪調整不能重迭或低於第一浪頂部；在下降走勢，第四浪調整不能重迭或高於第一浪底部。

  • 調整浪的對應規則

    1. 在波浪理論內，這原則並不是定律，市場告訴超過90%這原則可應用於第二浪及第四浪。

    2. 原則要求簡單第二浪對應複雜第四浪；反之，複雜第二浪對應簡單第四浪。

    3. 在外匯市場，對應原則可運用在調整比率上，若第二浪調整為第一浪x比率，第四浪便調整第三浪接近（1-x）比率。

    4. 在三角形調整內，浪A與浪B，及浪B與浪C，浪C與浪D，浪D與浪E型態上不相似。

  • 波浪比率及特徵

    1. 第一浪只是推動浪開始；

    2. 第二浪調整不能超過第一波浪起點；
    比率：2浪=1浪0.5或0.618

    3. 第三浪通常是最長波浪，但絕不能是最短（相對1浪和5浪長度）；這句話是不是有歧義比率：3浪=1浪1.618，2或2.618倍

    4. 第四浪的調整不能與第一浪重迭（楔形除外）；
    比率：4浪=3浪0.382倍

  • 基本理論

    平均線可說是最被廣泛應用的一種測市工具，亦是最易明白的一種理論，在現今測市法中，雖然有難度高低各異的各種方法，始終平均線仍是不可缺少的一環。平均線是以某一段時期的收市價相加起來再除以該段時期的日數，然後聯繫起來，與該日價位作出比較。用以預測未來市況之走勢。簡單而言，大市在平均線上表示後市向好，而大市在平均線下表示後市向淡。

    當平均線製成之後，本身亦會自成一種趨向。短期平均線通常會跟貼大市的升降而走動；中期平均線則會有自己獨立的走勢；而長期平均線可說是一種比大市慢但又是與大市走勢相若的趨勢。一般而言，短期平均線之走勢波動大，而長期平均線則較圓滑。

  • 入貨及沽貨訊號

    制定平均線的目的在比較目前的價位與平均線之間的關係，當價位穿越平均線時即成入貨或沽貨訊號。但必須以收市價來判斷，即是日之高低價若突破平均線時是不能單獨作準。而由於平均線在這一環節上之命中率較低，所以除收市價突破外，還需要翌日整日市況均在平均線上方可成訊號，亦有主張以突破百份之三至百份之五方可成訊號的理論。

  • 轉勢訊號

    在一般情況之下，平均線的製成有慢於大市的感覺，亦無形中形成了一股力量，如大市連升十天，那麼若製成一條十天平均線亦是會連升十天，但平均線則會在價位之下。這股無形的力量就叫做支持線，示意大市倘若下跌，會在此水平稍作停頓或反彈，其後之去向尚要結合其它分析理論。而當大市突破平均線而繼續下跌之時，也就是價位與平均線相交之時，亦是代表大市轉勢之時。而在價位下跌之後，初始的數天平均線仍是會繼續上升，這時表示跌勢尚未急劇，仍有回升的希望，及至其後如平均線本身仍由向上轉為向下之時，這表示熊市已經形成，大市若要恢復上升之勢尚要等一段時間。亦可以說熊市己進入第二期。反之，若大市由下跌轉為上升亦是同等看待。

    一般而言，越長期的平均線，在買入及沽出訊號方面則顯示出更為強烈及持久。突破長期平均表示後市會在同一方向行走一段較長時期。因此，短期平均線的指示性亦相應降低。平均線一般而言可分為短期、中期、長期及最長期的平均線。以香港股市的習慣，短期平均線則為十天，中期二十天，長期三十天，最長期為二百五十天。每星期有五個交易日，十天平均線是代表兩星期的走勢；二十天平均約代表一個月的走勢；五十天平均代表約兩個月的走勢。二百五十天則代一年之走勢。因一年有五十二個星期，撇除假期後則約為五十星期乘以五，得出二百五十日的平均線來。

    理論上，若價位突破十天平均線而向上升之時，代表一個短期的入貨訊號；但據一般經驗而言，這個訊號的命中率最低及最不可而中期的二十天線則有其應用之價值，在實用價值方面，五十天線則已經可代表牛熊之分界，而最長期的二百五十天線則只適用於宏觀經濟層面上，如政府用以制定政策或中央公積金決定投資策略時使用，對一般小市民來說，二百五十天線並無太大的實用價值。

  • 平均線的應用

    在應用平均線方面，除了平均線本身的走勢及與價格相交的關係外，各種平均線之間的相互關係亦有着舉足輕重的影響。當價位與三條平均線的走勢均向上時，表示走勢凌厲，而若平均線之間的距離越遠之時，表示支持力越強。這種情況通常在波浪理論中的第三浪出現，或是牛市的第二期出現。但當平均線之間的距離加大之時，示意大市己開始進入疲態，上升動力正在減弱中。

    由大市開始從低位回升起計算，單是觀看綜合平均線的距離己可捉摸大市的節奏。綜合平均線間的距離由窄變成闊示意大市上升動力正在增加中，直至變成最闊之時示意大市上升動力正在減慢中。而當綜合平均線間的距離由闊再變成窄之時，示意大市己進入牛皮待變的狀態，去向未明，等待穾破。

    當五十天平均線本身亦由上升轉為下跌之時，通常則是熊市的第二期或是波浪理論的爆炸浪。此時小投資者通常會作出一個錯誤決定，就是趁低吸納。但正確之方法則是順市加持短倉，切勿作螳臂擋車式的攔市。至於二百五十天平均線之應用法則是適用於極長期的投資。當價位下穿該線之時，表示大市已經來臨，而其下跌幅度達最高位與該線之間距離的一半，且其它平均線也下穿該線時，示意大市已經超賣，可以作長線投資的入市。反之，欲沽貨時也是一樣。

  • 結論

    單憑肉眼所觀看，平均線是一種落後大市的示意線，但每種圖表理論之所以繼續存在，必定有其特性。要充份利用其特性，就是將之用作非短線投資，即所謂由中短線至中線投資，則其應用範圍頗為廣闊。只要綜合其它理論自定一種買賣訊號，持之以恆，定有成果。

  • 上升三角形

    圖表理論，除可紀錄每日高低價位之餘，亦可將每日的價位連串起來成為不同的走勢或形態；這類情況便稱之為圖表形態。若將其配合技術線及陰陽燭以至波浪理論，則命中率會大大提高。由於圖表形態變化萬千，名稱亦多不勝數，以下僅列出常見之形態以供參考，由此再衍生出來之變化，在融會貫通之後亦不難明暸

    要預知大戶是否正在入貨，是否正在囤積居奇，最實用之方法就是觀察上升三角形。通常大戶入貨之時，是會用較低調的方法以減輕成本，因為若用高調方法，不斷買入時，則會將價位不斷扯高。因此大戶入市，通常是邊入邊出，即入了十數次後，價位便被扯高了，到時則要沽出三、四次，將價稍為壓低。以便有效地控制成本。

    若要從圖表方面來觀察，便會形成以上的圖形。首先要留意的是柱狀圖的形態，入貨之時收市價通常是較高的，但不一定是最高價收市，反正是屬於偏高收市。而到某一默認水平後，價位便開始回落。但回落之時其最低價則不會低前次的低位，怕會引起恐慌性拋售。

    如是者會往來數次，最高價則會在同一水平，最低價則會比前次之低位高；而歷時通常為兩至三個月。到收集完成之時，由於價位己被無形中被推高了，所以在自然情況之下，或是稍有任何小道消息的刺激，價位便會高於三角形之頂線。由於大戶入貨之時亦肯定有散家跟進，或自然入貨；在前兩三個月低位入貨者見價位稍高之時，散家通常會急忙出貨。又會將價位壓低回三角形之內。這種情況稱之為後抽。此時大戶見時機成熟，便動用最後之一筆資金一舉將該貨買起。由於經過一段頗長時期的收集，市面上便會出現貨源短缺，在供不應求的情況下，價位便會急速上升。

    在收集之目的達到後，便需要派發。而量度此升幅的最簡單方法則是以三角形之高度為其上升幅度。到達該點之時，亦會是上升結速之時

    要留意的是上升三角形是會在任何時期出現，但某些情況則其命中率會較高而某些情況其命中率則較低。通常在牛皮市況或價位從高位下跌後，其命中率則較高，而價位正在上升之時尤其是急升之時，所形成之上升三角形則不太準確。因為要經歷兩三個月的入貨，又要將價位推高推低，只會在牛皮市中才可成功造出來，若價位急升，大市壯旺時，便會形成好淡爭持，難以操縱任何形態。

  • 下降三角形

    凡事本質上均是相對的，有上升三角形自然會有下降三角形，但以命中率來說，上升三角形的命中率卻高於下降三角形，尤以在股票市場為甚。

    下降三角形基本上是大戶在派發時所做成的形態。當大戶沽貨低至某一水平之時，便會回購若干，使價位稍為回升，以便在更高的價位派發。如是者來回數回合之後，價位便會自然地輕微突破底線，再成一後抽，由於供過於求，價位便急速下跌。量度跌幅亦以三角形之高度為其下跌幅度

    上升三角形是出現在低位時期，市況淡靜，容易讓大戶控制價位；但下降三角形則應是出現在高位，市況壯旺，好淡爭持；難以讓任何人士所操控，所以形態很易被破壞而不能成形。若是在高位爭持之時，頭肩式則會比準確。

  • 變體三角形

    由於市場狀況瞬息萬變，實際上出現的上升或下降三角形都不會如上圖所示那樣成正體的規則形狀。上升的阻力線或下降的支持線通常都會參差不齊；而且更會缺少“因為”兩字吧？偏高或偏低而成不同形狀。頂邊偏高的稱之為變體上升三角形，底部偏低的稱之為變體下降三角形，而頂部同時偏低及底部同時偏高的稱之為對稱三角形或敏感三角形。

    A. 變體上升三角

    當上升三角形之頂部傾斜向上之時，表示上升之動力增大，其上升幅度亦會大於正體的上升三角形，量度之時除了以三角形之高度為標準外，還需要加上偏向上的直角高度，才可算為一完整的升幅預測。

    但必需要留意的就是當其上升之時，價位是升上偏上的頂線，而回落時，則必需穿越其直角線回到三角形之斜邊底部才算成形，若是碰到頂部後回落時不能穿越直角線，則此變體上升三角形便不能成形。可能變成上升通道或其它形態。

    B. 變體下降三角形

    當下降三角形之底部傾斜向下之時，表示下降之動力增大，其下降幅度亦會大於正體的下降三角形，量度之時除了以三角形之高度為標準外，還需要加上偏向下的直角高度，才可算為一完整的跌幅預測。

    但必需要留意的就是當其下跌之時，價位是跌至偏下的底線，而回升時，則必需穿越其直角線回到三角形之斜邊頂部才算成形，若是碰到底部後回升時不能穿越直角線，則此變體下降三角形便不能成形。可能變成下降信道或其它形態。

    C. 敏感三角形

    所謂敏感三角形是指上面的頂邊偏向下，而下面之底邊同時偏向上，成一種對稱形態。此時代表大市失去方向感，未知後市到底是會向上或向下。此種情況通常在牛皮市中出現。如敏感三角形向上突破之時，則代表後市向上，反之則向下。

    要留意的是變體的上升或下降三角形，若是破了上圖所示的實線便成為入貨或沽貨信號。但敏感三角形若是破了上圖中的實線，只可作為準備入貨或沽貨信號，因為其走向隨時可變成正體的上升或下降三角形。到破位之時，則可能進一步轉變形態成為正體的上升或下降三角形。到時再依照該圖形的正式入貨或沽信號行事即可。

  • 頭肩式

    頭肩式又稱頭雙肩，是一種命中率極高的形態；亦多數出現於最重要的轉角市。這是意味着一種甚為激烈的好淡爭持，而這種爭持是經歷最少三個回合的交鋒，最後某一方獲得壓性之勝利。翻譯為：最後以某一方獲得壓倒性的勝利結束。

    頭雙肩所紀錄的好淡爭持過程如下：首先好淡爭持中，好友先勝一回合，而淡友將其再推低至某一點。此時即形成了第一個浪，亦即是左肩。但此停頓點則頗為重要，日後便會成為判斷大市是否下跌的重點。此時好友再度大舉反攻，銳意將大市全面推高，而且更要高越上次的高位，那麼便成為頭頂。

    而此時淡友再行大力推低價位，但一到上次之低位時，好友即行全力托市，因為若低於上次低位之時，恐防會引起連串之拋售，到時想再將價位推高便甚難矣。而淡友因經歷了一段長時期的拋售，亦要稍為歇息，另一方面也是要銳意讓好友將價位推高，以便在高價沽貨，獲取更厚利潤。此時便會完成右肩。而由於淡友勢強，所以此次的高位卻不能超越前次的高位便開始下挫。到跌破頸線之時，無論是否有後抽，弱勢己成，會導致一發不可收拾的下跌場面。

    由於近來圖表派人士越來越多，所以頭雙肩便成為眾矢之的，眾人都留意着頭雙肩的頸線，即使有時不一定是那樣準確，只要跌破了頸線，便可引發羊群心理，以致出現恐慌性拋售。

    正如上升與下降三角形一樣，頭肩式亦有多種變體形態，包括了傾斜向上式，傾斜向下式，失敗式，單式及複式，陷井式以至用波浪理論來分析的頭肩式亦屬常見之態。都是從基本的頭肩式變化出來的，只要熟習其基本論，便可克複千變萬化的各種變體形態。同時亦要留意各種形態在各種市況出現的命中率，因某些形態，並不是萬試萬靈，而是要符合某些條件才算是成形的。

  • 倒轉頭肩式

    較常見的變體頭肩式就是傾斜向上式及傾斜向下式，其出現的情況甚至比正體的頭肩式還要多。而實上，任何一種圖表形態均是變體出現的機會比正體多。傾斜向上式表示大市向上動力較大，因此在下跌之時其跌勢亦緩慢，跌幅亦會縮少。而量度的方法則是以頭頂至頸線的高度“A”減去其傾斜度，即破位點“Ω”至原頸線“Σ”的高度，是為之“B”。其下跌幅度則為“C”，因而得出了“C = A - B”的公式來，亦即是走完“C”之後則可重新買入。 至於傾斜向下式則示意大市向下之動力較大，因此下跌之時跌勢會更為猛烈，下跌幅度會更大。其量度的方法則是以頭頂至頸線之高度“A”加上破位點“Ω”至原頸線“Σ”的高度，是為之“B”。兩者相加起來則為“C”，是為其下跌之幅度，因而得出了“C = A＋B”的公式來，即是走完“C”之後則可重新買入。但這兩種情況雖然“A”“B”“Ω”“Σ”均完全一樣，但“C”卻不相同。亦即是說當兩個頭肩式的高度及斜度均一致之時，則可比較其向上或向下之動力而可以計算出其跌幅來。 至於倒轉頭肩式的基調與上述是完全一致的，同樣是傾斜向上式表示上升動力大，向下式表示下跌動力大。計算方式與量度波幅與上述的剛好反過來看即是。而另一些變體的頭肩式形態則如下：

  • 防備走勢陷阱

    凡有走勢，就有陷井；皆因市場是千變萬化，雖然始終會循着某些軌跡而行走，但這些軌跡變化卻大於投資者所面對的或然率。通常投資者所面對只有兩個抉擇，升或跌。而價位所形成的形態遠遠多於兩種形態。因此，最重要的是要認識某種形態在某些情況下會變形，而某些情況下其命中率會較高。但只要有效地掌握其變化，要化解走勢陷井其實並不困難。
    根據各地金融市場在過往走勢及各個別商品之波動情況，其實是可悟出一破解走勢陷井的基本方法如下：

    要辨別走勢陷井，必需先暸解各種形態的成因。

    1. 正體頭肩式的形成是好淡爭持，通常是在大市壯旺、屢創新高之際才會出現，而此時命中率最高。

    2. 下降三角形的命中率較低，因為當大市壯旺之時，難以有足夠時間和空間讓大戶有系統地出貨，通常都是好淡爭持的頭肩式。

    3. 若大市從高位回落，表示進入淡靜狀態，買賣兩閑，難以再出現激烈的好淡爭持的頭肩式，此時其命中率則會較低，反而下降三角形若出現在此時則其命中率高，因為大市淡靜才有機會讓大戶有秩序地出貨。

    4. 而當價位從低位回升之後，再出現倒轉頭肩式則十居其八為走勢陷井，因為頭肩式的最大作用是用來預測轉角市的，而順勢而行之市況不必有激烈之好淡爭持。要預測順勢市況，以下還有很多方法。

    5. 倒轉頭肩式出現在大市急跌之後最為準確，或長期緩慢下跌亦可套用，因為在超賣之時，大市亦會在低位出現激烈之好淡爭持。

    6. 當大市經過一輪上升之後所出現的上升三角形，其命中率則會較低。因為上升三角形是紀綠大戶在低位入貨的狀況，大市只會被大戶越推越高，而大戶則不會在高位有秩序入貨，只會出現好淡爭持而己。

    7. 所以當大市回落後或牛皮時所出現的上升三角形，其命中率最高。
    只要徹底明暸各種圖形的成因，加以滆匯通，對其他情況下所出現的圖表陷井自能掌握。

    在實際情況中，一條信道是很難找出多個支持位或阻力位，而經常出現的就是下面有兩個支持位，而上面只得一個阻力位；或反過來看亦可以。此時假設出現了“A”及“C”兩個阻力位而只得一個支持位“D”，亦可以自行在“D”位自行拉出一條並行線是與“A－C”線平行的，亦可作為支持線。但條件是要價位直接由“A”回落到“D”點，如當中出現“B”點，此條上升通道則不成立，而是要再拉一條“B－D”線出來形成了一個喇叭形，此乃是上升通道的變體形態之一種。

    反之如在上升信道的頂位再出現相反情況，可以將“E－G”線連成再比對“D－F”線，則成為一倒轉喇叭型或稱金斗型。信道形成之後然才可加以各項的分析，但當它尚未完成之時，則要判斷它即將組成哪一種形態。如上圖中“B－H”線與“A－C”線最初期亦可算是一上升信道，但當價位再穿越支持線至“D”點，或繼續下去成為“D1”“D2”“D3”……之時，到底是信道變形抑或是突破了支持線，則仍要配合其它分析方法來衡量，切忌自行在下面再加上無數條支持線，否則便會中正走勢陷井，陷入萬劫不復的深淵中。

    上升通道的理論同樣可應用於下降信道下，基調完全一致的，只要反過來看即是。

    雙頂與雙底及其變體形態

    雙頂及雙底出現的機會可說比其它形態多，但由於其指示力量不夠，兼容易引起誤解，所以有必要詳細說明。當價位回落至A點再升上B點後再回落至C點，這時最容易誤導他人雙底己成形，尤其在G位回升之時更易角入陷井；其實一如其它形態，必需有某個突破點，才算成形，突破點在這裡則為D點。要量度其升幅則是以雙底高度由突破點D點起加上去便是。在上圖中，”A－C”線則成為了支持線，而”B－D”線則成為了阻力線；只要定出這兩條線來，就可以分別出其它的變體形態。雙頂形態亦是反過來看即可。

    但每種形態最難學習的就是其變體形態。此雙底形亦可作為信道形的變體，但是既非上升信道亦非下降信道，而是一橫行信道。如價位升至D點又不突破，而再回落的話，則成了正體的橫行信道，如再回落至支持線「A－C」之時則成為了三底形狀，要再等待上升突破阻力點才算成形。如超過了三個頂或底部的形態，則稱之為多頭形或多底形。此橫行信道亦稱之為密集區，在突破此橫行信道後，此密集區即成為下次上升或下降之支持位及阻力位。

    當雙底形成之前出現的形態，稱之為高低腳，而高低腳亦很容易變體成為上升或下降信道。更加可以應用波浪理論來分析，所以在其變體過程中，必需划出並行線來才可算是變體成功的信道形。更重要的是找出支持點及支持線，突破點及阻力線。如高低腳情況是先出現高腳後再出現低腳，即表示向下之動力較大；反之如先出現低腳其後再出現高腳，則上升動力較大。高低腳的升幅一如以往其它圖形一樣，在上升時將其高度加上高低腳之差異，而下跌之時則以高度減去高低腳之差異。

    圓頂及圓底形態

    當大市牛皮之時，最易出現的就是圓頂及圓底的態，此時最難辨別的就是其阻力位，支持位以至突破位。因為大市成弧形的形狀前進之時，甚麼位都會變得十分模，此時要尋求的就是雙殺或俗稱兩枝棍。

    雙底形態雖說是在牛皮市中出現，但通常牛皮市亦有分牛皮靠穩或牛皮偏軟。假若價位是由高位偏軟而下的話，此時則很難分出其突破點或阻力線，若是大市是牛皮靠穩至A點始回落後慢慢循着一弧形走勢偏軟再靠穩，則成了一個圓底形態，此時的入貨點就以突破之前A點的高位為準，但另一種較可靠的方法就是去尋找兩枝棍，此方法可應用於前者的牛皮偏軟市及後者的牛皮靠穩市，即如價位無突別指示時，不管是因任何個別股分之消息或大市的消息使其突然冒升，而有一枝棍是明顯比之前的任何一枝棍為長的，則視之為入貨信號。而沽出獲利之時則是在翌日尾市，即在第二枝棍比第一枝棍高的位置便可獲利。

    此種形態通常應用於二、三線股分，但第二枝棍必需獲利，即使日後再上升，亦不要再綣戀，否則便有可能會一世悔恨。至於若出現第三枝棍時，則要配合其它形態及分析而定之，因為隨時會變體而成為其它形態。

    以上任一種圖表形態均有其獨特之處，但切忌單獨運用，必需整體配合，尤其是要留意的就是它是否有機會變體成為其它形態，方可免墮入圖表陷井中。

  • 何謂陰陽燭

    目前的圖表理論中，有兩種劃線方法是最被廣泛應用的，分別是柱狀圖(Bar Chart)及陰陽燭圖(Candle Stick)。柱狀圖只是紀錄當天的最高、最低及收市價，一種平鋪直述的紀錄；而陰陽燭則會比較當天的開市價是高於或低於收市價，增加了一項即市的紀錄。由於有着短期的紀錄，所以會對未來市況的短期走勢有指示作用。而對大局或長遠走勢來說，兩種圖表所得出的結果是一致無疑的。

    無論是用計算器顯示圖表或用紙張人手紀錄，由於陰陽燭圖每天所佔的面積較大，所以只能在同一畫面中顯示出中期走勢；而由於柱狀圖較為壓縮，所以在同一畫面中能顯示較長期的走勢。至於圖表形態的應用，兩者亦完全沒有分別的。陰陽燭是七十年代日本期貨市場的衍生品。當時由於日本期貨市場的即市波動較大，於是引發了一種紀錄即市變化的陰陽燭線圖來。其後美國期貨迅即採用此方法，到如今無論外匯、黃金、期貨或股票均完全接受了這種畫圖方法。

  • 解剖陰陽燭

    陰陽燭的構成要的是白色或空心柱來顯示是日之收市價高於是日之開市價，而無需理會上日之價位與是日價位之比較。相反地，如是日收市價低於是日之開市價，則以黑色或實心來顯示。而兩種方法之共同之處，高低價位均以幼線來顯示。

  • 陰陽燭的種類

    A. 大陽燭

    只有一條空心柱，其上及下並無幼線顯示最高或最低價，因為是日開盤之後，並無下跌，而一直上升至收市，而又是以最高價收市。此種情形，示意一種強烈的升勢，意猶未盡，通常翌日仍有若干上升空間。

    B. 大陰燭

    只有一條實心柱，其上及下並無幼線顯示最高或最低價，因為日開盤之後，並無上升，而一直下跌至收市，而是是以最低價收市。此種情形，示意一種強烈的跌勢，意猶未盡，通常翌日仍有若干下跌空間。

    C. 抗跌槌

    在一個長期跌勢之後，而出現這種抗跌槌，表示跌勢即將完結，跌幅正在減少中。其尾巴越長則應驗越快。這種形態雖開市後一直下跌，但收市前卻出現反彈，只是力度不夠，未能收在開市價之上而己。如翌日再出現下述的上升槌則走勢更是明顯。

    D. 抗跌槌

    先跌後漲之走勢，是一種短期轉角市的先兆，買方是在好淡爭持中續漸轉強而非實時轉強。在長期跌勢後更為明顯，而尾巴越長則轉勢越近，這亦屬於單日轉向的形態之一。轉勢隨時會在翌日出現。

    E. 抗升槌

    在一個長期升勢之後，而出現這種抗升槌，表示升勢即將完結，升幅正在減少中。其尾巴越長則應驗越快。這種形態雖然開市後一直上升，但收市前卻出現回吐，只是力度不夠，未能收在開市價之下而己。如翌日再出現下述的下跌槌則走勢更是明顯。

    F. 下跌槌

    先升後跌之走勢，是一種短期轉角市的先兆，賣方是在好淡爭持中續漸轉強而非實時轉強。在長期升勢後更為明顯，而尾巴越長則轉勢越近，這亦屬於單日轉向的形態之一。轉勢隨時會在翌日出現。

    G. 高位爭持型

    此是一種代表好淡爭持甚為激烈的走勢，但始終仍是好友勝出。若此種情況是出現在長期的升勢後，則是一種升勢即將完結之兆。但仍會有一段距離，不會馬上出現，只不過是要處於戒備狀態。獲利貨應實時沽出，未有獲利之貨仍可等一段短時期；如新入市則應以即市為目標，不可作太長線之長倉。因為買家己開始受到考驗。通常此情況要配合較大的成交量才算成形。

    H. 低位承接型

    這是一種強弩之末的標準形態，若是接着第G項高位爭持形而出現。則未獲利之長倉仍應儘早沽出為宜。因為轉勢雖不一定在翌日出現，但機會甚大，有時無需經歷其他形態便實時轉入下降形態。此種形態通常稱為欲漲乏力形。是表示好友與淡友均經歷一種極高同時又極低價的爭持，距離亦相當之遠，即所謂過山車式的即市市況。由於前景仍未明朗，所以收市價無論怎樣，均在開市價附近。買賣雙方都甚為謹慎，不敢冒險。

    I. 小陰線

    這是一種強弩之末之標準形態，若接着第H項低位承接形而出現。則未獲利之短倉應儘早回補為宜。因為轉勢雖不一定在翌日出現，但機會甚大，有時無需再經歷其他形態便實時轉入上升的形態。此種形態通常稱為欲跌乏力形。是表示好友與淡友均經歷一種極高同時又極低價的爭持，距離亦相當之遠，即所謂過山車式的即市市況。由於前景仍未明朗，所以收市價無論怎樣，均在開市價附近。買賣雙方都甚為謹慎，不敢冒險。

    J. 十字線

    這是一種轉勢的最後通牒，無論是上升或下跌。是一種好淡爭持而收市價剛好與開市價相同。成交量亦在萎縮中。此種情況亦稱為懸崖勒馬，再不改變思想有可能會跌入萬劫不復之深淵中。

    K. 十字星

    十字線與十字星之共同之處均是開市價要巧合地與收市價相等。由於此種巧合不一定出現，所以轉勢時不一定會出現這兩種形態。此項十字星的出現則表示牛皮市即將來臨，如出現在牛皮市中時則表示牛皮市尚要運行一段時期。

    L. 一號球風型

    反攻前奏形，賣方雖然在大部份之市場時間中佔優，但最後買方仍能夠收復失地。賣方應如氣象台般懸掛出一號戒備信號，因這甚有可能是暴風雨的前夕。這種形態亦稱為T字形。

    M. 三號球風型

    反攻前奏形，買方雖然在大部份之市場時間中佔優，但最後賣方仍能夠收復失地。買方應如氣象台般懸掛出三號戒備信號，因這甚有可能是暴風雨的前夕。這種形態亦稱為倒轉T字形。

  • 相對強弱指數

    相對強弱指數通常簡稱為RSI， 目的是要分析目前是市勢是屬於強勢抑弱勢， 更重要的處就是此市勢何時終結， 其所預測的往往是中期走勢，如從浪底持貨至浪頂，獲利則十分豐厚。亦即是預測轉角市何時出現。命中率相當之高，因而最為廣泛所採用。基本理論是以某一時段之收市價來作為數據，從中選取出這個時段內上升的日子之總和以及下跌日子之總和，再尋求其比例。便可得知目前走勢是屬於強勢抑弱勢，而當相對強弱指數與市價出背馳之時，即示意轉勢在即。設計此指數之時，是首先將市場動量情況釐定在0至100之間，要明白RSI，先要明白RS，先列出其計算方法如下：

    RS = 9AU / 9AD
    RS - Relative Strength
    AU - Average Up
    AD - Average Down

    AU的計算方法:

    若當日之收市價C1較上日收市價C為高，則兩日收市價之間的升幅為C1-C，然後將九日內報升日子錄得的升幅相加，然後再將總數除以九日，便得到九日內平均的上升幅度。

    AD的計算方法:

    若當日之收市價C1較上日收市價C為低，則兩日收市價之間的跌幅C-C1，然後將九日內報跌的日子錄的跌幅相加，然後再將總數除以九日，便可計算到九日內平下跌幅度。因此RS是在期內大市上升幅度及下降幅度的對比，如上升幅度，大市勢越強；如下跌幅度大，則示意市勢越弱。將其代入0至100之間的動量時，可演變成以下公式：

    RSI = 100 - 100 / (1+RS)
    RSI = 100 - 100AD / AD+AU
    RSI = AU / AD+AU x 100

    要計算相對強弱指數，最基本是要定出一個時段來，可以用一時段來計算，亦可以用複式時段來計算，目前是要提高其命中率。通常用的都是九日和十四日這兩個時段。皆為神奇數字里有八和十三。取九及十四即是轉勢之時，冀望可以更準確地測出轉角市。九和十四天的指數各有其優劣點。九天指數波動較大，但示意的為較短期的走勢，所以其力量亦較少；而十四天指數時間較長，力量較大，但信號並不明顯。所以現時一般人採用的，均是複式指數，是將兩種指數混合來看。

    在動量零與一百之間，先找出一中位數即五十點，以此點來釐定大市之強勢或弱勢。當九天及十四天指數同時在五十點之上時，表示大市屬於強勢，而當兩者處於五十點之下時，表示大市屬於弱勢，因而可釐定買賣策略。但當兩者均高於八十點時，表示己進入超買區，隨時會有回吐之可能；而兩者均進低於二十點時，表示己進入超賣區，隨時會有反彈之可能。 以八十點和二十點來釐定超買或超賣，一般來說需要等待較長之時期，但易捉到浪頂或浪底，亦會錯過很多中小浪的機會。因此亦有人建議採取三十點和七十點作為超買及超區之釐定，由於其變動大，經常出現入貨或沽貨信號，所以命中率不及前者高。因此，此處仍是建議以二十點和八十點作為基數，要捉摸其中短期變化，尚有很多較有效之方法。

    在兩線均進入超買或超賣區之時，不就一定可以反手，還要等待這兩線相交，才算是正式的入貨或沽貨信號。而尚未相交之時，如九天線己到達零點或一百點，而十四天線順序己低於二十點或高於八十點亦可。到出現入貨信號時，先行買入，到兩線均超越五十點之時，則可沽出一半，到正式沽貨信號出現之時， 則要全部清貨離場。 而當中RSI的走勢， 亦可按照其它圖表形態畫出趨勢線或各種圖表型態來；然後再按該型態之信號進行買賣，即如破頭肩頂底線則沽貨，破頭肩底頂線則入貨；升破下降信道則入貨，跌破上升通道則沽貨等等。 相對強弱指數原本是要每天計算，每天劃的；但由於現在計算機信息發達，因此不但可顯示出真實時間的指數，以便即市買賣，而且還可以與其它技術線一起應用，以提高其命中率。

  • 關於隨機指數

    由於只使用一種技術線來預測大市，往往會出現所謂圖表陷井，於是類似的技術線會應運而生。通常隨機指數都會配合相對強弱指數而運用，因為兩者之應用方法頗為類似。但隨機指數則更為細微，因為有快版與慢版之分。通常快慢是用以預測短期走勢，而慢版則用以預測較長期之走勢。

    隨機指數是建基在收市會與整段上升或下降時段的關係。理論上，在上升市中收市會應接近整個時段的高位，而下跌市則應接近整個時段之低位。而計算之時間越長則命中率越高，但為了要等待訊之出現，可能亦會錯過一些短期之買賣信號。因此，實際應用起來，均是快版與慢版一起混合應用的。儘管慢版隨機指數是屬於改良型。原則上是以收市價偏高或偏低來預測未來走勢。基本計算公式是以最後的收市價減去時段內的最低收市價，再除以時段內最高的收市價減去時段內的最低收市價。茲列快版與慢隨機指數的出公式如下：

    但當信號出現次數少之時，其命中率自然會提高，同時所指示的走勢維持時間則會較長；反之，信號出現多之時，所指示的走勢維持的時間會較短，但會較實際，因為可指示出中期的入市或離市信號。而在實戰環境當中，中線的信號是最受歡迎的，因為除了本身的中線客之外，無論是長線客或短線客都隨時會轉變形勢追隨中線信號。況且，當快版與慢版隨機指數一起配合運用之時，其命中率亦是相當之高的。 至於其強勢及弱勢的分野同樣是五十點，可參照前面強弱指數的示意圖，理論上的應用方法是一致的，但實際上，唯一的分別是在圖表形態上。因為其信號出現的次數比較多，其走勢經常往返於八十點及二十點之間，所以大多數為近乎直線的走勢，很難造出如頭肩形或上升以及下降信道的形態來；所以在隨機指數以後述的MACD的圖形中，是無需理會這些形態的。率亦是相當之高的。

  • 離合線

    離合線全名為“移動平均線匯聚背馳指標”。與前述的相對強弱指數及隨機指數完全以另一種形態來表示的，其應用方法亦有甚大差異，所以三者若能配合應用，其命中率自然會大大提高。平均線實際上為一種雞肋線，其應用程度最為廣泛，但要最受批評的。主要由於其難度低及出信號時往往慢於大市，於是便有很多改良型的技術線如離合線等。計自方法是以當天的收市價減去上日的“指數移動平均線”（EMA），然後再將其差乘以一個固定的常數百分比，再加回昨日指數移動平均線。

    MACD = EMA(C,n1) - EMA(C,n2)
    訊號線 = EMA(MACD,n3)

  • 拋物線

    基于市场走势不会成绝对的直线或直角，而包含一定弧度，因此引用大自然的抛物弧度来应用在市场上。大自然上的所谓抛物线定义是由某一点所引伸出来的平面曲线，而该距离的点除以该距离的线是等于一。由此定律即可求出固定的参数。应用之时即是以某一日的高位减去时段开始时的转向点，即成翌日之转向点。再以翌日之转向点乘以固定参数，以预测目前走势何时完结，有如物体被抛落地面时其行程即告终止。

    抛物线基本上是一种时间与价位的系统。而此系统之最大特点是有足够时间去让市场反应。当一个走势开始之时，它不会马上转势，但当肯定此走势己形成后，反应才会出现，因而可避免走势陷井；到此走势继续进行之时，其点数则会跳得极为快速。所以这些点数不单止显示出价位因素，亦可显示出时间因素。当价位稍为回落之时，抛物线是不会回头的，直至大市掉头为止。所以应用抛物线之原则其为简单，只要抛物线向上，则持好仓；抛物线向下则持淡仓即可。而抛物点之距离亦是价位波动强弱的指示。

    基本公式如下
    SAR = SAR + AF (H-SAR)
    引伸公式則為
    SAR = SAR + AF (EP-SAR)

    SAR = Stop And Reverse
    AF = Acceleration Factor
    H = High
    EP = Extreme Price Point

    以下举一实例说明，当价位上升之时，开始时每天的抛物线是缓缓上升的，然后再急速上升，到最后之时其上升幅度是每天相等的，即变为只有价位因素而没有时间因素了。以图表来显示，若于第四天开始计算，第十天即失去时间因素。当第四日价位是50.00计算第五天的SAR则如下：

    SAR5 = SAR4 + AF (H4-SAR4)

    假设以第四日的最高价位减去第四日之SAR及将这距离乘以其加速之系数0.2，再将其加回至第四日之SAR，再代入公式中，则如下：

    SAR5 = 50.00 + 0.02(52.50-50.00)
    SAR5 = 50.00 + 0.02 X 2.50
    SAR5 = 50.00 + 0.05
    SAR5 = 50.05

    由此公式得出，第五日之SAR为50.05，其AF则为由0.02至0.20之一系列参数。而AF则每日以0.02之速度上升，因此第六日之SAR计算方法如下：

    SAR6 = SAR5 + AF(H5-SAR5)
    SAR6 = 50.05 + 0.04(53.00-50.05)
    SAR6 = 50.05 + 0.04 X 2.95
    SAR6 = 50.05 + 0.12
    SAR6 = 50.17

    如此类推，第七至十二日之SAR计算方法如下

    SAR7 = 50.17+0.06(53.50-50.17)=50.37
    SAR8 = 50.37+0.06(54.00-50.37)=50.66
    SAR9 = 50.66+0.10(54.50-50.68)=51.04
    SAR10 = 51.04+0.12(56.00-51.04)=51.52
    SAR11 = 51.52+0.14(55.50-51.52)=52.08
    SAR12 = 52.08+0.16(56.00-52.08)=52.71

  • 動力線

    市場蘊藏著無窮動力，再由這些無形的動力推動大市上升或下跌。動力指數就是以收市價來預測市場的動力到底是否完結。因為如大市上升之時，近期的升幅必定是高於以往升幅，但當升幅收窄，動力指數下跌之時，即使大市仍在上升，則表示升勢即將完結。計算方法是以當日之收市價減去若幹日前之收市價，而將其差額，正數或負數記錄在零在線或下。公式如下：

    M = Pt - Pt - n
    M = 動力指數

    Pt = 當日收市價
    n = 日數

    動力線的應用法與其它上述的技術線相異之處就是只用壹條技術線來預測走勢。相對強弱指數及隨機指數均有其上下限，其系數變化均在零與壹百之間；因此，有時候在接近零點或到達零點之時大市仍未反彈。但動力線則無此限制，定出零度線之後，隨時可以大幅偏下或大幅偏上，零度以下之負數或零度以上之正數均可以無限大。有些類似離合線，但不同之處就是離合線之去勢頗為極端，有時到達正壹千點或負壹千點大市仍未掉頭。而動力線則可說較為中庸之道。其實際應用法是以零度線作為分界，零度以上為牛市，零度之下為熊市。其走勢壹樣可比照圖表形態劃出上升信道或下降信道，頭肩式或雙頂雙底形態出來。主要是找出動力線與大市背馳之形態，即示意大市在數天內會掉頭。
