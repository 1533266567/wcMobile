  • 市價單

    止盈單

    止盈單僅當市場價格達到了指定價位或比之更高的時候被執行。止盈賣單是當賣價達到所要求的價位時被執行。止盈買單是當買價達到所要求的價位時執行。

  • 止損單

    當市場價格達到預設價位時自動執行買或賣的定單。當市場價格達到定單價格，止損單將成為市價單，以市場當時最有利且可交易價位執行。請謹記，如果市場沒有在定單的價格上交易，止損單可能由於市場跳空或滑點的影響而在定單設置的價格外執行。止損單將在定單價格被觸發後，按照市場下一個最有利且可執行的價位成交

  • 止損入場單

    在當前價位以上設置的買入定單，或在當前價位以下的賣出定單。該種定單可以在您認為市場將朝一個方向波動，而希望在某價位入場的情況下使用。

  • 止損平倉單

    這是一種設置以低於當前價格賣出（來平倉多頭頭寸），或以高於當前價格買入（來平倉空頭頭寸）的定單。止損定單是一種重要的風險管理工具。如果市場走勢不利於您，則您可通過為持倉頭寸設置止損定單限定您的可能損失。請記住，止損定單並不保證定單的執行價格 ,當止損單被觸發，定單將按照市場最有利且可行的價位執行。

    *如您需查看指定時間段的交易報告，請聯繫客戶專員

  • 移動止損

    移動止損定單可以自動調整觸發交易執行的止損點價位，使之朝着有利的市場方向以一定幅度移動。在市場價格向有利方向移動的前提下，移動止損功能可以擴大盈利空間，並且一旦市場價格朝不利方向發展，交易可以自動完結。當市場價格朝有利方向發展（在價格上漲時持有多頭頭寸，或在價格下跌時持有空頭頭寸），止損點價位跟隨着市場價格以一定幅度移動。如果市場價格朝不利的方向發展，則止損點價位保持不變，它和市場價格之間的差值也會縮小。如果市場價格繼續朝不利的方向移動，並最終達到了設定的止損價位，那麼交易將被執行。

    以上所有類型的定單均為＂周末前有效定單＂（GTF），也就是周末收市前訂單如果沒有被執行就會被取消。定單在被執行或取消前都是有效的。如果您手動清算一筆頭寸，必須取消所有與此頭寸相關聯的定單。須注意，條件定單有可能不能限制您的損失。

  • 定單範圍

    黃金的止盈或止損單必須離當前市場價最少200點以外的距離。

    白銀的止盈或止損單必須離當前市場價最少200點以外的距離。

  • 有效期

    周末前有效定單（GTF），也就是周末收市前訂單如果沒有被執行就會被取消。

    文傳國際在北京時間周六收市後到周一開市前不執行交易。重新開市後，以開盤價而非我們交易台關閉時可能出現的價格予投資者進行交易。

  • 盈虧計算

    盈虧計算公式：( 賣出價 - 買入價 ) x 合約單位 - 利息 - 傭金

    例子1：
    李先生經網上平台買入2手倫敦金，即200盎司， 投入保證金2,000美元，買入價為1,300美元，並於即日以1,330美元賣出2手倫敦金平倉，他的盈利計算方式是：
    投入保證金金額＝ 2,000美元
    淨盈虧＝ (1,330 - 1,300) x 200 = 6,000美元

    例子2：
    陳小姐經網上平台下單買入1手倫敦金，即100盎司，投入保證金1,000美元，買入價為1,250美元，其後於第二天以1,220美元賣出1手倫敦金平倉，她的虧損計算方式是：
    投入保證金金額＝ $1,000美元
    持倉利息＝ 1,250 x 100 x 1.5% x (1/360) = 5.21美元
    淨盈虧＝ (1,220 - 1,250) x 100 - 5.21（利息） = -3,005.21美元

  • 注意事項

    1. 周末及假期休市前半小時會調整點差，以系統顯示為準；行情波動大時，點差費用可能根據市況調整，報價以系統顯示為準。

    2. 如周五及假期收市時，賬戶保證金不足100%，系統將會把所有倉位做鎖倉（對沖）處理，以控制投資者的交易風險。

    3. 當賬戶的保證金比例不足20%時，系統會強制平掉所有的倉位。當價格波動大時，可能在低於20%的保證金比例時強制平倉。

    4. 周末或休市後所有未建倉的掛單及未執行的止損獲利設置將全部取消。

    5. 交易時間將因應國際金融假期而作出相關調整，詳情請留意本網站通告。
